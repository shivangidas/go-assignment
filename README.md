# go-assignment
GoLang assignment


## Todo Web app

How to start the app

    cd webApp
    go run .

The server should start at port 8080

Open http://localhost:8080/ on a browser

## Todo cli app

How to start the app

    cd cli
    go run .

For tests use `go test .` where test file is present
